Lis le fichier BRIEF.md et construis la landing page complète.

Etapes:
1. Crée un projet Next.js 14 avec TypeScript et Tailwind CSS: `npx create-next-app@latest . --typescript --tailwind --app --no-src-dir --import-alias "@/*"` (réponds "yes" aux prompts)
2. Installe les dépendances: `npm install lucide-react`
3. Construis la landing page complète dans app/page.tsx
4. Crée les composants nécessaires dans app/components/
5. Configure tailwind.config.ts avec les couleurs du design system
6. Lance `npm run build` pour vérifier que tout compile

La page doit être visuellement impressionnante: dark hero, sections alternées dark/light, cards avec hover effects, animations CSS simples.

Quand terminé, run: openclaw system event --text "Landing page datacenter-ai terminée — npm run dev pour tester" --mode now
